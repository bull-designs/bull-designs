A modern free responsive,
open source SCSS framework based on Flexbox

Download Zip Github Repository

Responsive

Responsive modifiers enable specifying different column sizes, offsets, alignment and distribution at xsmall, small, medium & large viewport widths.

            
              <div class="row">
                <div class="grid-xsmall-12
                            grid-small-8
                            grid-medium-6
                            grid-large-4">
                    <div class="box">Responsive</div>
                </div>
              </div>

Fluid

Percent based widths allow fluid resizing of columns and rows.

Simple Syntax

All you need to remember is row, column, content.

            
              <div class="row">
                <div class="grid-xsmall-12">
                  <div class="box">12</div>
                </div>
              </div>

Offsets

Offset a column

            
              <div class="row">
                <div class="grid-xsmall-offset-3 grid-xsmall-9">
                  <div class="box">offset</div>
                </div>
              </div>

Auto Width

Add any number of auto sizing columns to a row. Let the grid figure it out.

            
              <div class="row">
                <div class="grid-xsmall">
                  <div class="box">auto</div>
                </div>
              </div>

Nested Grids

Nest grids inside grids inside grids.

            
              <div class="row">
                <div class="grid-xsmall">
                  <div class="box">auto</div>
                    <div class="row">
                      <div class="grid-xsmall">
                        <div class="box">auto</div>
                      </div>
                    </div>
                  </div>
                </div>
              </div>
